# Portfolio Website

Personal portfolio website for Felix Hoffmann showcasing professional experience, education, and projects. Built as a modern static site with a focus on performance and accessibility.

**Live Site:** [https://felix-hoffmann.dev](https://felix-hoffmann.dev)

## Tech Stack

- [Astro](https://astro.build/) 5 - Static site generator
- [Tailwind CSS](https://tailwindcss.com/) 4 - Styling
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Alpine.js](https://alpinejs.dev/) - Lightweight interactivity

## Features

- Dark/light theme toggle with system preference detection
- Content-driven project showcase using Astro Content Collections
- SEO optimized with sitemap, meta tags, and OpenGraph support
- Responsive design with mobile-first approach
- Performance optimized with image optimization and prefetching
